Initial Website RECON & FOOT PRINTING:

  1. host command: Used to lookup IP addresses from domain names.
  1. nslookup: Performs DNS lookup.
  1. robots.txt file: Shows disallowed entries—files or folders you don't want search engines to index.
  1. sitemap.xml file: serves as a roadmap for search engines like Google. It lists all important pages, videos, and images on a website, telling search engines exactly where everything is located.
  1. Browser plugins OR Web extensions:
    1. Builtwith: Identifies and displays technologies used on any webpage you visit.
    1. Wappalyzer: Identifies technologies on websites.

  1. whatweb command : Web scanner that identifies technologies used by websites, including content management systems (CMS), JavaScript libraries, web servers, and other components.
  1. Downloading entire website: HTTrack WEBSITE COPIER. {sudo apt-get install webhttrack}

WHOIS Lookup:

  • Registrant Information: Name, organization, email address, phone number, and physical address of the domain owner. This information can be useful for social engineering or identifying points of contact.
  • Registration and Expiration Dates:
  • Creation Date: Helps assess how long the domain has been active, which can indicate legitimacy or potential use in phishing campaigns.
  • Expiration Date: Domains close to expiration may be vulnerable to takeover if not renewed.
  • Updated Date: Frequent changes might suggest active development or recent ownership changes.

Netcraft: Passively analyzes website infrastructure. Helps Fingerprint tech stack and map infrastructure without direct contact.

  • Hosting Provider and IP History
  • Web Server and Operating System (e.g., Nginx on Linux)
  • SSL/TLS Certificate information
  • Subdomains
