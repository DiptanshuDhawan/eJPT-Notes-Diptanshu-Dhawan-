Port Scanning With Auxiliary Modules

FTP ENUMERATION —>

  • auxiliary/scanner/ftp/ftp_version —> Checks the version of FTP service running.
  • auxiliary/scanner/ftp/ftp_login —> Try brute force for user and pass on ftp server.
  • auxiliary/scanner/ftp/anonymous —> Anonymous FTP Access Detection

SMB ENUMERATION —>

  • auxiliary/scanner/smb/smb_version —> SMB Version Detection.
  • auxiliary/scanner/smb/smb_enumusers —> SMB User Enumeration.
  • auxiliary/scanner/smb/smb_enumshares —> SMB Share Enumeration.
  • auxiliary/scanner/smb/smb_login —> SMB Login Check Scanner or for BRUTE-FORCING.

HTTP OR WEB SERVER ENUMERATION —>

  • auxiliary/scanner/http/http_version —> HTTP Version Detection.
  • auxiliary/scanner/http/http_header —> HTTP Header Detection.
  • auxiliary/scanner/http/robots_txt —> HTTP Robots.txt Content Scanner.
  • auxiliary/scanner/http/dir_scanner —> Directory Brute-Forcing.
  • auxiliary/scanner/http/file_dir —> Interesting File Brute-Forcing.
  • auxiliary/scanner/http/http_login —> HTTP login form Brute-Forcer.
  • auxiliary/scanner/http/apache_userdir_enum —> This module is often used in penetration testing to gather information about potential users on a web server.

MySQL ENUMERATION —>

  • auxiliary/scanner/mysql/mysql_version ——> MySQL Server Version Enumeration.
  • auxiliary/scanner/mysql/mysql_login —> MySQL Login Utility. also used for Brute force.
    • try to set username to root and then brute force password to get root credentials.
  • auxiliary/admin/mysql/mysql_enum ——> Need credentials to run this module.
    • Allows simple enumeration of Server provided proper credentials to connect remotely.
  • auxiliary/admin/mysql/mysql_sql ——> Need credentials to run this module.
    • Allows us to execute SQL queries on the server if we have credentials.
  • auxiliary/scanner/mysql/mysql_schemadump——> MySQL Schema Dump.
